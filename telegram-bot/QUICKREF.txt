Problem: Bot won't start
Fix:     Check .env file has TELEGRAM_BOT_TOKEN set

Problem: "Import aiogram could not be resolved"
Fix:     pip install -r requirements.txt

Problem: "Failed to connect to API"
Fix:     Start API server: ./bin/api-server

Problem: No notifications
Fix:     Check /list, restart monitor, verify address active

Problem: Invalid address format
Fix:     Address must start with 'T' and be 34 chars
